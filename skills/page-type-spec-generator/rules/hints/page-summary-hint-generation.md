# page-summary-hint 生成规则

## 目标

从 xlsx 输入生成 `page-summary-hint.md`。

`page-summary-hint.md` 只承载页摘要阶段的补充规则。

## 输入

1. 当前 type 对应的页摘要素材

## 输出

1. `page-summary-hint.md`

## 总规则

1. 只写页摘要阶段的组织偏好与重点。
2. 不得写标签取值规则。
3. 不得写块组装策略。
4. 不得写块摘要模板。
5. 无素材时输出空模板。

## 应承载的内容

1. 页摘要优先呈现的信息
2. 页摘要组织顺序偏好
3. 页摘要中的特殊表达要求

## 不应承载的内容

1. 标签定义
2. 块组装规则
3. 块摘要规则
4. 具体模板句库

## 生成步骤

### Step 1：判断是否有素材

规则：

1. 若无素材，输出空模板
2. 若有素材，再继续生成

### Step 2：提取页摘要规则

动作：

1. 提取页级重点信息
2. 提取组织顺序偏好
3. 提取页级表达约束

## 写法规则

### 必须做

1. 每条规则必须直接服务页摘要生成
2. 规则必须可执行

### 禁止做

1. 不得重复 `value_hint`
2. 不得写块组装策略
3. 不得写块摘要规则
4. 不得无素材时灌水

## 空模板规则

若无素材，输出：

```md
# {类型名} · 页摘要生成指导

<!-- TODO: 当前设计稿未提供专门的页摘要规则 -->
```

## 质量检查

生成后必须检查：

1. 是否只承载页摘要规则
2. 是否误写取值规则
3. 是否误写组装或块摘要内容
4. 若无素材，是否为空模板

## 失败信号

以下情况说明生成失败：

1. 把标签取值规则写进 `page-summary-hint.md`
2. 把块摘要规则写进 `page-summary-hint.md`
3. 把组装策略写进 `page-summary-hint.md`
4. 无素材却生成大段泛化废话
