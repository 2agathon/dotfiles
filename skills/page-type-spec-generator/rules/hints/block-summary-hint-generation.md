# block-summary-hint 生成规则

## 目标

从 xlsx 输入生成 `block-summary-hint.md`。

`block-summary-hint.md` 只承载块摘要阶段的补充规则。

## 输入

1. `块摘要构造和输出组装`
2. 与块摘要直接相关的 `采集和转化规范`
3. 与块摘要直接相关的 `缺项处理`
4. 与块摘要直接相关的 `冲突处理`

## 输出

1. `block-summary-hint.md`

## 总规则

1. 只写块摘要阶段的内容偏好与兜底规则。
2. 不得写标签取值规则。
3. 不得写块组装策略。
4. 不得写页摘要模板。
5. 无素材时输出空模板。

## 生成条件

以下任意一列有实质内容时生成实质内容，否则输出空模板：

1. `块摘要构造和输出组装`
2. 与块摘要直接相关的 `采集和转化规范`
3. 与块摘要直接相关的 `缺项处理`
4. 与块摘要直接相关的 `冲突处理`

## 应承载的内容

1. 摘要必须出现的重点内容
2. 摘要组织顺序偏好
3. 常见遗漏项提醒
4. 块摘要的特殊表达要求

## 不应承载的内容

1. 标签定义
2. 块组装顺序
3. 页摘要写法
4. 具体占位模板句库

## 来源列到输出层级映射

1. `块摘要构造和输出组装` 是主体内容
2. `采集和转化规范`、`缺项处理`、`冲突处理` 只作为补充节

## 生成步骤

### Step 1：判断是否有素材

动作：

1. 检查是否存在块摘要级素材

规则：

1. 若无素材，输出空模板
2. 若有素材，再继续后续步骤

### Step 2：提取摘要重点

动作：

1. 从 `块摘要构造和输出组装` 中提取摘要重点与组织偏好

规则：

1. 该列内容优先级最高

### Step 3：提取摘要兜底规则

动作：

1. 从与摘要直接相关的 `采集和转化规范`、`缺项处理`、`冲突处理` 中提取块摘要阶段仍需保留的约束

### Step 4：组织输出

动作：

1. 按摘要重点、格式偏好、常见遗漏提醒分节输出
2. 主体内容必须先于补充节输出

## 写法规则

### 必须做

1. 每条规则必须直接服务块摘要生成
2. 规则必须可执行
3. 规则应尽量短

### 禁止做

1. 不得重复 `value_hint`
2. 不得写块组装策略
3. 不得写页摘要模板
4. 不得给出大段占位模板句库

## 空模板规则

若无素材，输出：

```md
# {类型名} · 块摘要生成指导

<!-- TODO: 当前设计稿未提供专门的块摘要规则 -->
```

## 质量检查

生成后必须检查：

1. 是否只承载块摘要阶段规则
2. 是否误写取值规则
3. 是否误写组装策略
4. 若无素材，是否为空模板
5. 是否保持了“主体内容 + 补充节”的结构

## 失败信号

以下情况说明生成失败：

1. 把标签取值规则写进 `block-summary-hint.md`
2. 把组装策略写进 `block-summary-hint.md`
3. 把页摘要写法写进 `block-summary-hint.md`
4. 无素材却生成大段泛化废话
