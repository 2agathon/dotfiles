# continuation-hint 生成规则

## 目标

从 xlsx 输入生成 `continuation-hint.md`。

`continuation-hint.md` 只承载前页续写判定阶段的规则。

## 输入

1. 当前 type 对应的续写判定素材

## 输出

1. `continuation-hint.md`

## 总规则

1. 只写续写判定阶段的结构与边界规则。
2. 不得写标签取值规则。
3. 不得写块组装策略。
4. 不得写摘要模板。
5. 无素材时输出空模板。

## 应承载的内容

1. 哪类结构单元可跨页
2. 哪类内容不得参与续写
3. 续写终止条件
4. 版式重复段识别规则

## 不应承载的内容

1. 标签定义
2. 块组装规则
3. 摘要写法
4. 具体模板句库

## 生成步骤

### Step 1：判断是否有素材

规则：

1. 若无素材，输出空模板
2. 若有素材，再继续生成

### Step 2：提取续写规则

动作：

1. 提取可跨页单元
2. 提取不可跨页内容
3. 提取终止条件
4. 提取版式重复段规则

## 写法规则

### 必须做

1. 每条规则必须直接服务续写判定
2. 规则必须可执行

### 禁止做

1. 不得重复 `value_hint`
2. 不得写块组装策略
3. 不得写摘要规则
4. 不得无素材时灌水

## 空模板规则

若无素材，输出：

```md
# {类型名} · 续写判定提示

<!-- TODO: 当前设计稿未提供专门的续写规则 -->
```

## 质量检查

生成后必须检查：

1. 是否只承载续写判定规则
2. 是否误写取值规则
3. 是否误写组装或摘要内容
4. 若无素材，是否为空模板

## 失败信号

以下情况说明生成失败：

1. 把标签取值规则写进 `continuation-hint.md`
2. 把块组装规则写进 `continuation-hint.md`
3. 把摘要规则写进 `continuation-hint.md`
4. 无素材却生成大段泛化废话
