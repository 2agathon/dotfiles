# 海报布局规范 v4

## 整体风格
- 底色：#faf7f2（温润纸白）
- 主色：#1e1c1a（深墨）
- 强调色：由 groupPersonality 动态决定
  - 热闹型 → #c8402a
  - 专业型 → #2a5f8f
  - 温情型 → #4a7c59
  - 混合型 → #7b5ea7
- 字体：Noto Serif SC + Space Mono
- 宽度：520px，高度不限

---

## 区块顺序

1. **封面** — 大标题（AI画像）+ 数字条 + 群友共识
2. **人物志** — 弹性九宫格，最多9人
3. **本期故事** — 报纸头条式，序号+标题+叙述+引用
4. **对话节奏** — conversationSessions 时间轴，合并沉默
5. **数据** — 热力图 + 内容构成，双列
6. **活跃排行** — 头像+条形+百分比
7. **本期对白** — 气泡样式，左右交替，彩色背景区
8. **本期之最** — insights 六项卡片，3列网格
9. **深度** — 词云/最长对话链/问题榜/话题转折
10. **本期彩蛋** — 冷知识 + 沉了的消息
11. **情绪收尾** — AI一句话，居中大字
12. **底部** — 品牌标识 + 日期

---

## 人物志弹性布局规则

| 人数 | 布局 |
|------|------|
| 1 | 1列 |
| 2 | 2列 |
| 3 | 3列 |
| 4 | 2×2 |
| 5 | 3+2 |
| 6 | 3×2 |
| 7 | 4+3 |
| 8 | 4×2 |
| 9 | 3×3 |

原则：不让尾行出现孤零零单个成员。

---

## 时间轴规则

- 数据来源：conversationSessions（不按天，按对话段落）
- 段落间沉默 ≥1小时才显示「沉默 N 小时」
- 总行数 >20 时，合并消息数 <3 的小段落
- 沉默显示格式：「沉默 X 天」或「沉默 X 小时」
- 合并段落显示：「另 N 段对话 · 共 X 条」

---

## 气泡对白规则

- 不使用 box-shadow（html2canvas 不支持）
- 用 border + background-color 做层次
- 左气泡：border-radius 2px 12px 12px 12px
- 右气泡：border-radius 12px 2px 12px 12px
- 背景色用成员颜色 + 18 透明度（hex）
